    - 增加代码注释
    - 参考文献设置移除宏包
    
- **1.0.0** (2022-08-17) 初始版本
    - 间距格式调整

    - 兼容 overleaf

    - 附录

## 使用
请参看 GZUthesis.tex 文件

## 编译
编译 说明文档
```bash
xelatex GZUthesis.tex
```
编译示例文档
```bash
xelatex template.tex
```
